Document Object Model (DOM) is a programming interface for HTML.JavaScript DOM is the API that JavaScript uses to communicate to static HTML elements, making them more functional. The DOM is a tree-like representation of the contents of a webpage using nodes instead of elements. These nodes represent HTML elements, and they can be manipulated and re-rendered.

Some purpose of DOM:-
1. Element : generic class from which all objects that represent elements descend.
2. Window : represents a window containing a DOM document. The document interface mentioned above refers to the document inside this window.
